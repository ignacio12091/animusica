REPOSITORIO DE ANIMÚSICA

Requisitos para iniciar la app:
  - Clonarse el repositorio actual (https://github.com/ignacio12091/animusica) - BRANCH "entrega"
  - Clonarse el repositorio con el código del servidor (https://github.com/ignacio12091/animusica-server) - BRANCH "entrega"
  - Crear una base de datos de postgreSQL y utilizar el script de la creación de las tablas y los inserts presentados en el informe. Es       importante aclarar que se debe tener corriendo pgAdmin con estas configuraciones para que funcione la aplicación web.

Una vez con los requisitos cumplidos, ejecutar el comando "npm install" dentro de las carpetas de ambos repositorios y así instalar las dependencias.

Luego, corremos el comando "npm start" ambos proyectos (app de React y NodeJS) y después de varios segundos debería mostrarse la página web.
